# 葵花宝典

## 操作命令

`Hexo`初始化

    hexo init #类似于git init

`Hexo`添加文章

    hexo new blog-name

`Hexo`删除文章

`Hexo`删除文章只需要删除./source/\_posts 文件夹下对应的`.md`文件后，重新编译即可

`Hexo`清除缓存

    hexo clean

`Hexo`生成（暂时可以理解为编译

    hexo g(enerate)

`Hexo`服务预览

    hexo s(erver)

`Hexo`部署

    hexo d(eploy)

## 实用工具

`Hexo`热部署
&emsp;&emsp;如果想要更新代码后试试刷新调试界面，可以尝试使用 `hexo-browsersync` 这款工具，安装方式也只需要使用 npm 安装，安装后即可使用：

    npm install hexo-browsersync --save
